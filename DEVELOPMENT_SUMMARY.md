## 一、项目概述

趋势中军交易系统是一套面向个人投资者的 A 股量化决策辅助工具。核心理念：**用多因子模型识别主线板块 → 在主线中锁定中军标的 → 按均线信号执行买卖 → 三级熔断控制风险**。

系统定位为"决策辅助"而非"自动交易"——所有交易信号由系统生成，实际执行由投资者在券商 App 手动完成。

## 三、技术架构

```
Web 层:  FastAPI + Jinja2 + HTMX + Chart.js
   ↓
Workflow 层: Weekly → Daily → Monthly 流程编排
   ↓
Engine 层:  Scanner(Screener/Selector) + Monitor + Position + Risk
   ↓
Repository 层: SQLite 7 个领域 Repository
   ↓
Provider 层:  Market / Financial / Announcement (akshare)
```

**设计原则**：
- 单用户、单进程、零外部中间件
- `python main.py --serve` 一键启动
- Engine 纯函数可单元测试、Provider/Repository 依赖注入

## 四、核心功能模块

### 4.1 主线识别 (B-1/B-2)

- **板块扫描器**：遍历全市场 ~200 个行业/概念板块，四维加权评分（趋势强度 35% + 相对强度 25% + 资金确认 25% + 梯队完整性 15%）
- **主线确认器**：四项持续性验证（连续跑赢/产业逻辑/非天量/PE 分位）
- **退潮预警器**：跑输大盘/涨停萎缩/成交萎缩/中军长阴 四信号

### 4.2 中军筛选 (C-1/C-2)

- **五维评分模型**：市值规模 + 流动性 + 均线结构 + 量价健康度 + 基本面质量
- **均线监控器**：A 级买点（MA10 回踩企稳）、B 级买点（板块共振启动）、减仓/清仓信号

### 4.3 仓位风控 (D-1/E-1)

- **仓位管理器**：四重硬约束（总仓 70%/单票 30%/单板块 35%/最低现金 30%）
- **三级熔断器**：L1 个股 -8%止损 / L2 日回撤 -3%禁开 / L3 月回撤 -10%强制空仓

### 4.4 流程编排 (Workflow)

- **周度**：更新板块 → 扫描 → 确认主线 → 筛选中军 → 持久化
- **日终**：更新行情 → 均线信号 → 退潮预警 → 净值快照
- **月度**：绩效报告（胜率/盈亏比/板块归因/买点归因）

## 六、已知限制

1. **M2 阶段代理指标**：ThemeSelector 的"产业逻辑"和"PE 分位"验证使用近似指标（trend+fund 双确认），标注 FIXME(M3) 待真实数据源
2. **Scanner volume_ratio**：当前测量板块时序放量而非市场占比，标注 FIXME(M3)
3. **无券商 API 对接**：交易指令需人工在券商 App 执行（设计如此）
4. **单机部署**：无云同步、多设备支持
5. **网络依赖**：依赖 akshare 实时拉取东方财富数据，离线环境不可用
